This is a simple To-Do-List Application.

Tech used
  -HTML
  -CSS
  -Javascript
  -Node.js
  -Express
  -MongoDB

The Index.html is the frontend file for the application.
The server.js is the backend for the application.

All the modules to be used was included in this repo.

To run the project use (> node server.js) in the terminal or command prompt.

In case if it didn't work, use (> npm install express mongodb mongoose path body-parser ) .

This application provides the basic functions of a CRUD application. It can create Data( Create Tasks ) , Read Data ( Read and Store Tasks ) , 
Update Data ( Update/ Edit Tasks ) and Delete Data ( Delete Tasks ).

For now this task is specifically made as the Minor project as a CRUD application for the Corizo Web dev internship.
